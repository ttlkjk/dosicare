Product Overview
솔루션 개요
스마트한 통합 QA 플랫폼
Linac QA Hub는 방사선 치료 품질관리의 파편화된 데이터를 하나로 연결합니다. 수동 입력의 오류를 줄이고, 대규모 데이터를 규격화된 리포트로 즉각 변환하여 의료물리사가 본질적인 분석 업무에 더 집중할 수 있는 환경을 제공합니다.
📋
데이터 중점 관리
점검표의 디지털화를 통해 장기적인 트렌드 분석과 안정적인 DB 보관이 가능합니다.
🔬
실시간 빔 분석
측정 직후 Flatness, Symmetry 등 핵심 빔 지표를 즉각 계산하여 시각화합니다.
🔗
자동화된 연동
주요 Dosimetry 장비(IBA Blue Phantom, PTW, BeamCheck, QuickCheck 등)로부터 데이터를 자동 로드하여 휴먼 에러를 방지합니다.
📄
원클릭 리포팅
기관 맞춤형 양식으로 PDF, Excel 보고서를 자동 생성하며 디지털 서명을 지원합니다.
✦ 주요 분석 및 관리 기능
  • 일간 MPC 점검 자동 결과 파싱
  • 주간/월간 점검표 수식 자동 계산
  • 연간 종합 성능 지표 비교 분석
  • T-Log 궤적 로그 분석 기반 MLC 검증
  • Winston-Lutz 아이소센터 정밀도 측정
  • Picket Fence MLC 선형도 평가
  • StarShot 회전축 정확도 시각 분석
  • PDD · Profile 빔 데이터 오버레이

일간/주간 QA

  • 주요 에너지별 출력(Output) 기록

월간 QA

  • Dosimetry / Mechanical 정밀 분석
  • 외부 장비 데이터 자동 전송
  • 에너지별 편차 트렌드 모니터링
  • Tolerance 기준 초과 시 차등 강조
Yearly

  • TG-142 기준의 포괄적 항목 준수
  • 연단위 성능 변화 비교 및 문서화
  • 종합 평가 의견 기술 및 서명
